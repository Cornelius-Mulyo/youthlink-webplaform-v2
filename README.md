# YouthLink Opportunity Platform

YouthLink is a Flask-based web application that allows users to discover and share youth opportunities such as internships, scholarships, workshops, and volunteer programs.

Users can also upload documents related to opportunities and manage them directly through the platform.

---

## Features

- Add new opportunities with title, description, and category
- Upload files related to opportunities
- View uploaded files
- Download uploaded files
- Delete uploaded files
- Search opportunities by title, description, category, or filename
- Filter opportunities by category

---

## Tech Stack

- Python
- Flask
- HTML
- Bootstrap
- Jinja2 Templates
