# YouthLink Opportunity Platform

YouthLink is a Flask-based web application designed to help users discover and share youth opportunities such as internships, scholarships, workshops, and volunteer programs.

The platform allows users to add opportunities, upload related documents, and manage those files directly through the interface. It demonstrates core full-stack web development concepts including file management, search functionality, and dynamic content rendering.

---

## Features

- Add new opportunities with a title, description, and category
- Upload files related to opportunities
- View uploaded files directly from the platform
- Download uploaded files
- Delete files from the system
- Search opportunities by title, description, category, or filename
- Filter opportunities by category

---

## Tech Stack

- **Python**
- **Flask**
